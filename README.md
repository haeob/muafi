# muafi
> muafi(floaty-image)
- 一个浮动图片工具☁️🥰🎵

## 功能
- 让图片在你的屏幕上动起来！（当然也可以静止不动😋）
- 可以自定义图片/GIF、图片碰撞音。（这会很有意思）
- 自定义图片的数量、大小、透明度。
- 自定义音量的大小。
- 运动、静止模式。（静止模式时可以随意拖动图片位置）
- ...(更多功能请期待✨)
